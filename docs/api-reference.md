# CoParse API Reference

Base URL: `http://localhost:8000`

Interactive docs: `http://localhost:8000/docs` (auto-generated by FastAPI)

---

## Health Check

### `GET /health`

Returns API health status.

**Response:**
```json
{ "status": "healthy" }
```

---

## Search

### `GET /api/search/`

Semantic search over contract clauses.

**Query Parameters:**
| Param | Type | Required | Description |
|-------|------|----------|-------------|
| `q` | string | Yes | Natural language search query |
| `role` | string | No | User role (default: "general"). Options: general, intern, tenant, contractor, employee |

**Response:**
```json
{
  "query": "non-compete clause",
  "role": "intern",
  "results": [
    {
      "text": "Employee shall not engage in competing business...",
      "clause_type": "non-compete",
      "risk_level": "high",
      "explanation": "This clause restricts your ability to work for competitors for 2 years, which is unusually long for an internship.",
      "similarity_score": 0.92
    }
  ]
}
```

## Analysis

### `POST /api/analysis/`

Upload and analyze a contract document.

**Request:** `multipart/form-data`
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `file` | file | Yes | Contract document (PDF, DOCX, TXT) |
| `role` | string | No | User role (default: "general") |

**Response:**
```json
{
  "filename": "internship_agreement.pdf",
  "role": "intern",
  "total_clauses": 15,
  "risks": [
    {
      "clause_text": "Intern assigns all intellectual property...",
      "risk_type": "risky",
      "severity": "high",
      "explanation": "This clause transfers all IP rights, including work done on personal time.",
      "market_norm": "Most internship agreements limit IP assignment to work done during business hours using company resources."
    }
  ],
  "summary": "Found 3 high-risk clauses and 2 unusual clauses in this internship agreement."
}
```

## Contracts

### `GET /api/contracts/`

List all stored contracts.

**Response:**
```json
{
  "contracts": []
}
```

### `GET /api/contracts/{contract_id}`

Get a specific contract by ID.

**Response:**
```json
{
  "contract_id": 1,
  "message": "Contract detail"
}
```
